The Project

Great Music Taste

ShakespeareAI is a custom trained GPT-2 model that learned the writing style from Shakespeare.

Interactive

ShakespeareAI will write lines based off of any prompt you provide it.

Timeline

ShakespeareAI was completed by teenagers over the course of three weeks this summer!

Extensions

Some extensions to this project include deploying it to the web, building a discord bot, or ShakespeareAI interact with more writers!
